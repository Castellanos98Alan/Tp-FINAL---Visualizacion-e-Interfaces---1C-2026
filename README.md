## 📋 Descripción

Este repositorio contiene el trabajo práctico final desarrollado para la materia **Visualización e Interfaces**.
El proyecto consiste en una página web orientada a ayudar a los vecinos del partido de **Morón** a reportar diferentes problemas de la vía pública y del barrio, como baches, fallas de alumbrado, acumulación de basura u otras situaciones que requieran atención municipal.
El sitio permite iniciar reclamos, visualizar reclamos existentes, consultar un mapa interactivo y acceder a distintas secciones informativas. La propuesta busca brindar una interfaz clara, moderna y fácil de usar para mejorar la comunicación entre los vecinos y el municipio.
